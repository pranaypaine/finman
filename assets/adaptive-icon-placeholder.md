# Adaptive Icon Required (Android)

Please replace this file with your adaptive icon foreground:

- **File name**: `adaptive-icon.png`
- **Size**: 1024x1024 pixels
- **Format**: PNG with transparency
- **Purpose**: Android adaptive icon (foreground layer)

## Design Tips:
- Design should fit within center 66% circle
- Outer 18% may be cropped on some devices
- Use transparency for the icon shape
- Background color is set in app.json (currently white)

## Resources:
- [Android Adaptive Icons Guide](https://developer.android.com/guide/practices/ui_guidelines/icon_design_adaptive)
- Test with: https://adapticon.tooo.io/
