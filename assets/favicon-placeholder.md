# Favicon Required (Web)

Please replace this file with your favicon:

- **File name**: `favicon.png`
- **Size**: 48x48 pixels (or larger, will be scaled down)
- **Format**: PNG or ICO
- **Purpose**: Browser tab icon for web version

## Design Tips:
- Simple, recognizable at tiny size
- High contrast
- Avoid fine details

Once you have your favicon, replace this file with `favicon.png`
